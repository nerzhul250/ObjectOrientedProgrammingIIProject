ARCHIVOS DE GUARDADO----------------------------------------------------------------------------------------------------
los archivos de guardado están en la carpeta ---DATA--- del proyecto
Carpeta---EstadoGraficadora--- Es la carpeta en donde se encuentran los archivos de guardado del estado de la graficadora
Carpeta---HistorialSistema--- es la carpeta en donde se encuentra el archivo en donde se guarda el historial de sistemas
Carpeta---MatricesGigantes--- es en donde se coloca el archivo de texto para las matrices. MATRIZ11 es la matriz 1
                              MATRIZ2 es la matriz 2. Los nombres de los archivos deben ser iguales al reemplazar las matrices
DOCUMENTOS---------------------------------------------------------------------------------------------------------------
Capeta docs
carpeta--diseo de pruebas--- es la carpeta en donde se encuentra todo el diseño de pruebas
carpeta---postMortem--- es la carpeta en donde se encuentra el postmortem
capeta--requerimientos funcionales--- se encuentra el archivo con los requerimientos funcionales
carpeta---specs--- diagramas de clases y dashBoard
carpeta---instrucciones--- todas las instrucciones de cómo usar el programa para que tenga un buen funcionamiento
